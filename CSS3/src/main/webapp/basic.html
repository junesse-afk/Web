css basic

선택자 {속성:속성값;}

1. css 대표 선택자

  • tag선택자 - tag 그대로를 선택자로 활용. 해당 tag전체가 바뀌므로 강제적용, 자동적용
  • id선택자 - tag에 id를 추가하고 그 값을 선택자(#id값)로 활용. id가 중복적용 불가하므로 해당 소스를 재활용 불가하여 주로 구조적인 용도에 사용.
  • class선택자 - tag에 class를 추가하고 그 값을 선택자(.class값)로 활용. class는 중복 적용 가능, 조합이 가능하므로 주로 디자인적인 용도에 사용.

2. css 적용 방법

  • External CSS (외부 CSS) - CSS소스를 외부 파일로 독립시켜 html문서와 연결한 형태 (head안에 link href="*.css" rel="stylesheet") 로 적용. 유지보수에 유리, 통일성 있는 디자인 적용 가능.
  • Internal CSS (내부 CSS) - CSS소스를 html문서의 head안에 추가하여 적용. 해당 html에만 종속적으로 적용 됨.
  • Inline CSS (인라인 CSS) - CSS 소스를 tag뒤에 직접 추가하여 적용. 소스가 길어지고 유지보수에 불리, 통일성 있는 디자인 적용 불가. 본문만 수정 가능한 상황일 때 불가피하게 사용. CSS적용 우선 순위는 가장 높음.
