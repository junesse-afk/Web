CSS 선택자

  • 태그선택자 : tag 그대로 선택자로 활용, 해당 tag전체가 바뀌므로 강제적용, 자동적용
  • #아이디선택자 : tag id를 추가하고 그 값을 선택자(#id값)로 활용, id가 중복적용 불가하므로 해당 소스를 재활용 불가하여 주로 구조적인 용도에 사용
  • .클래스선택자 : tag에 class를 추가하고 그 값을 선택자(.class값)로 활용 class는 중복 적용 가능하므로 주로 디자인적인 용도에 사용

CSS 삽입방법

  • 외부 CSS External CSS : css소스를 외부 파일로 독립시켜 문서와 연결한 형태로 적용.
  • 내부 CSS Internal CSS : css소스를 html 문서의 head안에 추가하여 적용. 해당 html에만 종속적으로 적용됨
  • 인라인 CSS Inline CSS : css소스를 tag뒤에 직접 추가하여 적용
