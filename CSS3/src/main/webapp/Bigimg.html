background - 요소의 배경

  1. background-color:color; 배경 색상 지정
  2. background-image:url("이미지경로"); 배경 이미지 지정

background-image의 속성들

  • background-repeat: ; 배경 이미지 반복 지정
    • repeat
    • repeat-x
    • repeat-y
    • no-repeat
  • background-position: ; 배경 이미지 위치 지정
    • left/center/right top/center/bottom
    • Xn Yn
  • background-size: ; 배경 이미지 크기 지정
    • cover
    • contain
    • Xn Yn
  • background-attachment: ; 배경 이미지 스크롤여부 지정
    • scroll
    • fixed
    • local
