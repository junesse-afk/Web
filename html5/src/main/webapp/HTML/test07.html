3. input type="checkbox" 다중 선택형 객관식. 체크박스

복습이 필요한 과목은?

공간절약형 객관식, 펼침목록(드롭다운리스트)

등교에 걸리는 시간은?
다중 선택형 박스가 되려면 select multiple

형태만 박스가 되려면

select size="" > 대신 다중선택안됨

그룹화 박스가 되려면

optgroup label="">

4. input type="radio" 단일 선택형 객관식. 라디오
