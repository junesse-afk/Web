iframe - 독립된 영역

<iframe src="보여질 내용(경로)" width="" height="">iframe 미지원 브라우저 사용자용 메시지</iframe>

iframe 안에 보여질 수 있는 종류

  • URL
  • 웹페이지(*.htm/*.html)
  • 이미지(*.gif/*.jpg/*.png)
  • 동영상(*.webm/*.ogg/*.mp4)
  • 텍스트파일(*.txt)

iframe을 구별할 수 있는 name의 값을 주고 그 name의 값을 target의 값으로 지정

<a href="" target="abc"> <a>

<iframe src="경로" name="abc"> <a>
