HTML 수업 첫번째

HTML = Hyper Text Markup Language의 약자.
웹페이지의 구조를 만드는 표준 언어.

html 명령어(Tag)이루어짐.
Tag는 일반적으로 각 괄호 안에 들어가고 시작Tag와 종료Tag의 한 쌍으로 구성 됨.
원칙적으로 대소문자 구별은 없으나 소문자로 작성.
html 문서의 파일 저장은 *.htm 나 *.html 로 지정.

html의 필수 Tag들

html	---------	html문서의 시작을 선언
	head	--------- 	머리말의 시작을 선언
title	---------	제목을 선언
/title	--------- 	제목 끝을 선언

/head	--------- 	머리말의 끝을 선언
body	--------- 	본문의 시작을 선언
br	--------- 	줄바꿈을 선언
/body	--------- 	본문의 끝을 선언
/html	---------	html문서의 끝을 선언
