form-2 (html5 요소들)

1. input type="time" 시간(오전/오후 시 분)을 지정하는 폼요소


2. input type="date" 날짜(년월일)를 지정하는 폼요소


3. input type="datetime-local" 날짜와 시간(년월일 오전/오후 시 분)을 지정하는 폼요소


4. input type="week" 날짜(년주)를 지정하는 폼요소


5. input type="month" 날짜(년월)을 지정하는 폼요소


6. input type="number" 숫자값을 지정하는 폼요소

min/max로 입력값의 최소/최대를, step로 증감 간격을 지정

명 입니다 (min="0")
명 입니다 (max="10")
명 입니다 (step="5")

7. input type="range" 범위내의 값을 지정하는 폼요소


8. input type="tel" 연락처(전화번호)를 지정하는 폼요소

모바일브라우저에서 가상키패드가 숫자형식으로 바뀔 수 있고 전화걸기 가능

9. input type="email" 연락처(이메일)을 지정하는 폼요소

브라우저에서 알맞은 이메일형식 인지 체크 해 줌(@유무, @이후 작성 여부)

10. input type="url" 연락처(웹주소)를 지정하는 폼요소

브라우저에서 알맞은 웹주소형식 인지 체크 해 줌(프로토콜 http://, https://유무)

11. input type="color" 색상값을 지정하는 폼요소


12. input type="search" 검색용 상자를 지정하는 폼요소

서로 관련된 폼요소들을 그룹화 하기 - fieldset

fieldset으로 폼요소들을 묶어주고 해당 fieldset의 제목(주제)은 legend로 묶어서 나타냄.

필수



html5 추가된 input 속성들

1. 자동포커싱. 입력대기상태가 됨 - autofocus="autofocus"(autofocus)

2. 자동 완성 기능을 켜거나 끔 - autocomplete="on/off"

3. 입력예시나 설명을 지정 - placeholder="..."

4. 필수입력요소로 지정 - required="required"(required)

5. 읽기만 가능한 요소로 지정 ( 입력값이 전송 됨 ) - readonly="readonly"(readonly)

6. 사용 불가능한 요소로 지정 ( 입력값이 전송 안 됨 ) - disabled="disabled"(disabled)
